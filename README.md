
# CRUD

CRUD is an acronym widely used in computing. It refers to the four basic operations for manipulating persistent data:
- Create: Add new data.
- Read: View or search existing data.
- Update: Modify existing data.
- Delete: Erase existing data.


## Tech Stack

**Client:** Sveltekit + TailwindCSS


## Run Locally

Clone the project

```bash
  git clone https://github.com/Gautierpicon/CRUD
```

Go to the project directory

```bash
  cd CRUD
```

Install dependencies

```bash
  bun install
```

Start the server

```bash
  bun run dev
```
